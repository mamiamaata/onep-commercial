# ONEP - Système de Gestion Commerciale

Système complet de gestion commerciale avec base de données JSON Server.

## Déploiement
1. Fork this repository
2. Go to https://my-json-server.typicode.com/
3. Use your GitHub username/repository
4. Your API will be available at: `https://my-json-server.typicode.com/your-username/onep-commercial`

## Fonctionnalités
- Gestion des clients
- Gestion des commandes  
- Module de recouvrement
- Tableau de bord interactif
